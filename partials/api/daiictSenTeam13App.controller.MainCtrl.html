MainCtrl
controller in module daiictSenTeam13App

Description

Controller for the route /. It checks authenticates the user and redirects to the appropriate page.

Methods

  • goToRegister()

    Redirects user to registration page.

    Returns
    undefined

    Does not return anything.

  • goToReset()

    Sends the password to user's email.

    Returns
    undefined

    Does not return anything.

  • initMaterial()

    Initializes Materialize components.

    Returns
    undefined

    Does not return anything.

  • login()

    Gets the email and password from scope. Makes a login request to Firebase. If successful, checks for temporary password or calls redirectUser. In case of login error, raises appropriate toats.

    Returns
    undefined

    Does not return anything.

  • redirectUser()

    Fetches the authentication data from local storage of the browser. Identifies the type of user i.e. faculty or student by making an API call to /users. Also sets attaches userType variable to $rootScope.

    Returns
    undefined

    Does not return anything.

  • validate()

    Validates user input on main page.

    Returns
    undefined

    Does not return anything.
